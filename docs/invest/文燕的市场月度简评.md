# 市场月度简评
by 文燕

[[TOC]]

## 2025年3月
3月，**A股**呈现“前高后低”态势，上半月延续春季攻势，下半月受业绩披露预期、资金获利回吐等因素影响，主要指数普遍回调。上证指数全月下跌约1.2%，创业板指跌幅超3%，科创50指数跌逾5%。

**港股**方面，科技股受美股中概股波动及A股情绪传导影响显著回调，但部分消费与金融股相对稳健。

另一方面“东升西落”，**美股**在对经济滞涨的担忧中大幅回调，纳斯达克跌逾8%。

**国内债券**市场逐步企稳，稳健的客户仍可配置债券。但因为利率水平已是比较低的水平，未来债券收益很难像去年收益那么好，因此需降低收益预期，并寻求其他增厚收益的策略。

**黄金**方面，3月继续上扬突破3100美元/盎司，主要受地缘政治风险、美联储降息预期及全球央行增持驱动。

展望二季度，首先是4月2日的特朗普的关税政策是否落地受到全球瞩目。在关税政策上，中国可能受影响最大，但准备也最充分。因此我更倾向于关税风暴之后，很可能是**A股回暖、港股休整、美股修复**的趋势。当然，进入财报月，需关注财报验证，以及政策落地节奏。

资产配置方面，当前A股风险溢价处于近10年87.5%分位，港股估值修复至历史中枢，股债性价比仍偏向**权益资产**。

## 2025年2月
2月，**A股**主要指数均实现反弹，大盘再次回到了3300点上方区域。**港股**更是在科技股的带领下，大幅超越A股涨幅。

春节是市场情绪的分界点 ，DeepSeek在春节期间的持续发酵、春晚上的机器人扭秧歌、哪吒2跻身全球票房前十动画片票房第一等等，无不彰显了我们的实力。在科技上，我们并没有被美国拉下太远，尤其是DeepSeek的出现，是在受到制裁情况下，开发出了跟ChatGPT差不多性能但成本降低了九成的工具。在商业机器人板块，突然意识到我们的机器人全球领先，宇树科技的全球占有率超过50%，再想到我们的新能源车在全球的竞争力，无人机领域的优势等等，无不提升了市场的信心，哪吒的出圈和”我命由我不由天“更是强化了信心。

不过，**科技股**在2月上涨过大，短期内面临调整可能。但对于未来的科技股仍是看好，AI的发展像是打开了一扇门，未来各行各业与AI的应用发展可能是值得期待的。昨天听一个大佬分享会，关于科技的看法，虽然现在已不是科技股的最好卖点，后面的震荡也会加剧，但AI的发展像是互联网初期，未来行业应用方面发展空间更大，而且中国人更擅长应用方面。

另一方面，虽然2月大涨带动大盘上涨，但行业分化严重，在科技股和传媒股大涨之时，其他行业涨幅不明显 ，A股如要继续向上，也期待其他行业的回升。也期待下周的两会政策有进一步的超预期正常。
对于A股港股，短期可能会震荡，但目前估值仍在历史中低水平，仍是看好建议配置，可考虑红利低波+科技股的哑铃结构。不过资产分散配置是我们要坚持的，要根据每个人的风险偏好和资金流动性要求来分配资金，投资股票市场的资金建议是不影响日常生活可以较长时间不用的资金。

**债券**市场，2月进入调整阶段，可能很多朋友的银行理财也在下跌。利率下行到目前阶段，未来债券收益肯定是不如去年和前年的，但债券的配置意义是因为债券有票息，日常不怎么波动、作为稳健防守的资金去处。今年我们要降低债券的收益预期，但仍是看好的。

**黄金**方面，2月再创历史新高后本周有些回落，在国际形势没有明显变化的情况下，部分央行增持黄金的动力仍在，但确实2023年和2024年黄金上涨太快，建议已持仓获利的可继续保持，新进入的资金需谨慎。

投资路上，最大的确定就是市场的不确定性，我们保持仓位保持在场，做好资产配置更从容面对资产的短期波动。最后祝朋友们投资顺利，生活幸福，财源滚滚[發][红包]

## 2025年1月
1月的市场可能对于很多人来说，体感比较差，主要是由于在12月阴跌之后，都在期盼1月能有开门红，结果A股在2号-3号大跌超4%的情况下开局。不过我是认为，25年开年，在承受特朗普上台后对国内的压力、国内政策的空窗期的时候，1月已经比较坚挺。3号后市场最低跌至3140.95（上证），随后开启反弹，说明去年9月的反弹在10-12月的震荡筑底后，3150的底部还是很夯实。

今天是龙年最后一个交易日，本期待着最后一天的春节红包，但在deepseek周末发酵的情况下，以科技为主的创业板大跌，而市场交投清淡，最后市场收跌。全月市场主要指数涨跌概况如下表所示。另一方面，港股在特朗普上任后行情好于A股，尤其是恒生科技1月大涨。

关于投资，股票、债券、商品仍是我们最主要的投资资产类别。

股票市场的性价比更优，25年建议增加股票类基金的比例配置。

债券市场的稳定性和收益的确定性更好，波动小，但市场利率下行较快，对于债券的收益预期要下调。

商品的投资，黄金1月反弹较多，已逼近前期高点，在逆全球化趋势下，黄金的避险需求仍可做适当配置，建议比例5%左右。其他商品参与难度较高，可考虑用CTA策略参与。
