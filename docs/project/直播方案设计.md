# 直播方案设计
## PC服务器，双摄像头

## 树莓派拉流


## 方案说明

**方案1**
```mermaid
flowchart LR
  风景 --> device_1[**萤石监控摄像头**]
  猫们 --> device_2[**PC摄像头**]
  device_1[萤石监控摄像头] -. Wifi+rtsp .-> software_1[**B站直播姬**
  组合画面]
  device_2[PC摄像头] -- USB --> software_1
  software_1[**B站直播姬**
  组合画面] --> B站直播间
```

**方案2**
```mermaid
flowchart LR
  萤石APP --> device_1[**萤石监控摄像头**
  控制旋转、音频、跟踪]
  猫们 --> device_1[**萤石监控摄像头**
  控制旋转、音频、跟踪]
  萤石监控摄像头 --> 交换机 : Lan + rtsp
  交换机 --> 树莓派4B : ffmpeg拉流+转码+推流
  树莓派4B -[dashed]-> B站直播间 : rtmp
```


配置：

